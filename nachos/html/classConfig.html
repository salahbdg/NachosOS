Detailed Description

Defines Nachos hardware and software configuration.

Used to avoid recompiling Nachos when a change in the configuration is needed (turning on/of debug flags, changing memory size, etc.). There is a default value for every configuration parameter when not specified in the configuration file (see file config.cc)
