class  ACIA_sysdep
 

Detailed Description

Data structure to simulate an Asynchronous Communicating Interface Adapter.

The system dependent ACIA provides emissions and receptions of char via a simulated serial link. Il will be implemented by using sockets through the net. An emission and a reception can be parallelized (full duplex operation). All the accesses to the sockets are already defined in the module sysdep.h, so they will just have to be renamed.
