Detailed Description

Defines an entry in a simple translation table.

Each entry defines a mapping from one virtual page to one physical page. In addition, there are some extra bits for access control (valid and read-only) and some bits for usage information (use and dirty).

Depending on the 'swap' bit:

  • swap == false: location, in terms of BYTES, from the beginning of the executable file, or -1 for anonymous mapping
